About This Tracker

This application uses the MTA's GTFS-Realtime feed to track subway trains throughout the NYC transit system. The data is updated approximately every 30 seconds.

Select a train line above to view the current positions of trains on an interactive map.

Data provided by MTA. This application is not affiliated with the MTA.
